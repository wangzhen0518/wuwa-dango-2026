# 小组赛规则

1. 每个小组进行上半场和下半场两次比赛，进行下半场时，继承上半场结束时各团子的位置。
2. 每次比赛当有团子到达终点时，比赛结束。
3. 每个小组 6 进 4 晋级淘汰赛，剩下的 2 个团子进入谢幕赛。
4. 团子按照下半场结束时与终点的距离计算排名，离终点越近，排名越靠前。

# 小组赛分组

B 组：千咲团子、莫宁团子、琳奈团子、爱弥斯团子、守岸人团子、珂莱塔团子
C 组：奥古斯塔团子、尤诺团子、弗洛洛团子、长离团子、今汐团子、卡卡罗团子
