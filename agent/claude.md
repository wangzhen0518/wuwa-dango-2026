# 任务说明

这是一个比赛项目，我希望按照规则模拟比赛，看各个团子的胜率如何，你需要实现模拟比赛的程序。

这是各文件的内容：

- `./比赛规则.md`: 描述了比赛规则。
- `./赛道信息.md`: 描述了比赛的赛道信息。
- `./团子技能.md`: 描述了每个团子的技能。
- `data/赛程信息.json`: 提供了每场比赛的参赛团子、黑马值和比赛结果（如果比赛已经结束），"结果" key 对应的 array 按顺序为比赛第一名到最后一名，你需要在运行时读取这个文件。

如果对比赛规则或者任何逻辑不确定，请不要自己猜测，而是向我询问。

# 需要实现的功能

1. 模拟比赛，通过大量模拟计算出各个团子的胜率，即每个团子属于每种排名的概率，用表格形式表示，写入 `result/{相应比赛名称}-{运行时间}.csv` 文件，模拟过程应当高效、快速。
2. 根据团子名次的概率分布和黑马值，计算为每个团子应援的期望收益。
3. 根据给定的部分比赛结果，模拟后续的比赛，比赛结果以文本文件的形式提供。
4. 给定模拟结果，能够以 tui 的形式可视化地演示一遍比赛过程。

其他需要实现的功能请参考每个具体文件。
